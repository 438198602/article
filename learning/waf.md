# 网络应用防火墙（WAF）

* 防护 DDoS 攻击，对特定的 IP 地址限连限速；
* URI 黑名单和白名单，与 IP 黑白名单类似，允许或禁止对某些 URI 的访问；
* IP 黑名单和白名单，拒绝黑名单上地址的访问，或者只允许白名单上的用户访问；
* 过滤请求报文，防御“代码注入”攻击；
* 过滤响应报文，防御敏感信息外泄；
* 审计日志，记录所有检测到的入侵操作

注：注意` 木桶效应 `

注：使用成熟的WAF产品

注：https只能保证通信链路的安全，但它不能保护客户端和服务器。因此对于` DDOS `和`代码注入`等攻击无效。

注：WAF主要使用的是模式匹配和数据过滤的方法，所以会大量消耗cpu资源。
